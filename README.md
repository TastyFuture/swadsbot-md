## For Termux
```bash
$ pkg upgrade && pkg update[Download Script](https://blog.zeeoneofc.xyz/2022/02/alphabot-md.html)
$ pkg install bash
$ termux-setup-storage
$ cd /sdcard
$ cp -r Alphabot-Md $HOME
$ cd
$ cd Alphabot-Md
$ bash install.sh
```
